# PyTempGPU
A lightweight real-time NVIDIA GPU temperature monitoring tool for Linux terminals using NVML and Python.
> [!IMPORTANT]
> Need **python3**, **NVIDIA** **GPU** + **Drivers** and **Required** **Python** **Packages**(nvidia_smi, pyfiglet) to be installed.

- `PyTempGPU.py` is the main application.
- `README.md` is the how to use application.
